## 下载

### Docker

```
docker pull apachecn0/flygon-algo-zh

# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-algo-zh
flygon-algo-zh <port>
# 访问 http://localhost:{port} 查看文档
```
